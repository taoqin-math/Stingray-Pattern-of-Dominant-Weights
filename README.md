# Stingray Pattern of Dominant Weights

This repository contains the source code, scripts, and figures associated with the research paper: **"Stingray Pattern of Dominant Weights"**.

## Overview

The purpose of this repository is to ensure reproducibility and provide access to the computational tools used to generate the results and visualizations presented in the paper. You are free to use, modify, and build upon the materials provided here.

## Requirements

* Python, Sagemath (My version is Sagemath 10.7 and Python 3.13.1)
* $\mathfrak{sl}_4$ interactive figures are drawn using plotly
